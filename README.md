# Kingsdale Emporio | Digital Flagship Placeholder

**Engineered by Morse Data Technologies**

## Overview
This repository contains the source code for the "Coming Soon" placeholder page for **Kingsdale Emporio**, a premium lifestyle and custom apparel brand based in Cape Town, South Africa. 

Designed to serve as a digital anchor at `kingsdalesocks.co.za` while the full e-commerce platform is under development, this lightweight static site employs a "Dark Luxury" aesthetic. It balances high-end prestige with the brand's vibrant, modern streetwear character.

## Features
* **Bespoke UI/UX:** A striking, responsive layout utilizing a deep color palette to emphasize the brand's gold-accented logo.
* **Premium Typography:** Integration of Google Fonts (*Cinzel* for regal headings, *Montserrat* for clean copy).
* **Frictionless Communication:** An integrated, floating WhatsApp Business bubble with a pre-populated intent message to drive immediate client conversion.
* **Performance Optimized:** Zero dependencies. Built purely with semantic HTML5, modern CSS3, and lightweight Vanilla JavaScript to ensure instantaneous load times.
* **Brand Consistency:** Custom favicon and Apple Touch Icon configurations included.

## Technology Stack
* HTML5
* CSS3 (Custom Variables, Flexbox, CSS Animations)
* Vanilla JavaScript
